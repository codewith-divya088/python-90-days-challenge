# 📊 Student Record Manager

## 📌 Description
This is a simple Python-based project to manage student records.  
It allows users to add student details, calculate results, and view records.

## 🚀 Features
- Add student name and marks
- Automatically calculates Pass/Fail
- View all student records
- Simple menu-driven program

## 🛠️ Technologies Used
- Python (Core Concepts)
- Loops
- Conditional Statements
- Lists

## 🎯 Learning Outcome
- Improved understanding of loops and conditions
- Learned how to manage data using lists
- Built a real-world mini project

## 💡 Future Improvements
